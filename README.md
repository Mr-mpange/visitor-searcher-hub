# Find Place

A platform to discover and book accommodations, rides, and events near you.

## Live Site

[https://mr-mpange.github.io/visitor-searcher-hub/](https://mr-mpange.github.io/visitor-searcher-hub/)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth + database)
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npx gh-pages -d dist
```
